🔒 YouTube API Key Guide

How to Find Your YouTube API Key

You need a YouTube Data API v3 key to connect your channel. Here's a step-by-step guide:

1

Go to Google Cloud Console

Visit console.cloud.google.com and sign in with the Google account linked to your YouTube channel.

2

Create a New Project

Click the project dropdown at the top, then click "New Project". Give it any name (e.g., "My YouTube Channel") and click Create.

3

Enable YouTube Data API v3

Go to APIs & Services → Library. Search for "YouTube Data API v3" and click Enable.

4

Create Credentials

Go to APIs & Services → Credentials. Click "Create Credentials" and choose "API key". Your key will be generated immediately.

5

Copy Your API Key

Copy the API key shown. It will look like: AIzaSyXXXXXXXXXXXXXXXXXXXXXXXX. Paste it in the form below.

⚠ Security Note

We recommend restricting your API key to YouTube Data API v3 only in Google Cloud Console. This ensures your key cannot be used for anything other than displaying your YouTube content. We never store your key publicly — it is used only to fetch your channel information.
